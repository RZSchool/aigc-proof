# Signature Profile

## Purpose

Describe the intended interoperable signature profile.

## Current decision

Manifest normalization is planned as RFC 8785 JCS, hashing as SHA-256, local signing as Ed25519, and the signature container as COSE_Sign1. None is implemented in Phase 0.

## Not decided

Key-discovery, algorithm identifiers, and signature verification policy are unsettled.

## Next

Review established standards and publish test vectors before implementation.
