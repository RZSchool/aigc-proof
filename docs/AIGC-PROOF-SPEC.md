# AIGC-Proof Specification

## Purpose

Set the V0.1 public protocol direction.

## Current decision

V0.1 aims to record generation processes, input/output digests, final actual prompt snapshots, event chains, local signatures, and offline verification. It does not automatically determine originality.

## Not decided

Exact field semantics, signing input, and compatibility guarantees are drafts.

## Next

Align the specification with schemas and deterministic test vectors.
