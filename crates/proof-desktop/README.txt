AIGC-Proof Desktop Preview 0.2.0
================================

平台
----
Windows x64（单文件原生预览版）。双击 AIGC-Proof.exe 即可启动；不需要 Cargo、
Node.js、开发服务器、账号或网络连接。

使用顺序
--------
1. 选择或输入一个尚不存在的工作区路径，填写可选项目名，然后初始化；也可打开已有工作区。
2. 选择输入、输出、参考、许可或其他资产文件并添加。
3. 填写事件类型和 JSON 对象载荷，记录事件。
4. 选择新的 .aigcproof 输出路径并封装。
5. 选择证明包，分别执行“验证”和“仅检查元数据”；验证后可保存 JSON 报告。

隐私与能力边界
--------------
程序完全离线运行，不上传文件，不包含遥测，不调用官方服务，也不依赖私有仓库。
它只验证 .aigcproof 包内部的 Manifest、资产摘要和事件哈希链是否自洽。

它不验证创建者身份，不包含数字签名或可信时间戳，不判断原创性、版权、作者身份、
所有权或法律效力，也不是公证、版权登记或官方认证服务。

安全行为
--------
工作区、证明包和报告均拒绝静默覆盖。验证沿用公开 v0.2 核心的 ZIP 路径、重复条目、
大小写冲突、符号链接、条目数量、单文件大小、总解压大小和压缩比限制。

已知限制
--------
- 这是面向 Windows x64 的预览版，界面为固定尺寸原生 Win32 布局。
- 大文件操作会显示忙碌进度，但本预览版尚不提供中途取消。
- “检查元数据”不会验证完整性；必须使用“验证证明包”获得 valid/invalid/error 结果。
- 未进行代码签名，Windows 可能显示未知发布者提示。
