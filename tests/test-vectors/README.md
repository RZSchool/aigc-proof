# AIGC-Proof 0.2 deterministic vectors

These small UTF-8 vectors cover SHA-256, the RFC 8785 primitive-serialization and UTF-16
property-order examples, and the first event hash. Files named expected contain exact bytes or
lowercase hexadecimal text followed by one newline for readability. Implementations hash the
canonical JSON bytes without that explanatory newline. Security tests generate valid, tampered,
duplicate, ZIP64, special-file, traversal, and resource-limit packages dynamically.
